Software

Visual Studio code

Visual Studio code is een code-editor die wij gebruiken voor het schrijven van code. Dit programma maakt het schrijven van codes makkelijker door de toegevoegdde extensies die je kunt gebruiken.

GitHub

Github is een platform dat softwaredevelopers de vaardigheid geeft om hun projecten op te slaan en om samen te werken aan hetzelfde project.

Figma

Figma is een onderwerptool die vaak wordt gebruikt voor het maken van ontwerpen die worden gepresenteert aan de gebruiker voordat de dienst word aangeboden.
